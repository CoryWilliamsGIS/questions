Web Application User Guide

 

Prerequisites:

        Clone the following Github Repository: https://github.com/CoryWilliamsGIS/questions

        Clone the following Github Repository: https://github.com/CoryWilliamsGIS/server

        Navigate to and run the httpServer.js file in the terminal by typing:
node httpServer.js &

        Navigate to and run the Phonegap Sever in the terminal by typing:
phonegap serve

        For a more detailed guide on the above steps, please follow the web application installation instructions in the README.md file of the following Github Repository: https://github.com/CoryWilliamsGIS/server

Creating a New Question:

        Locate the question form at the bottom of the application screen.

 

        Populate each field of the question form, indicating the correct answer with the check buttons below. Unless you know the exact desired coordinates, leave the location fields blank for now.

 

        To choose the location of the question, click the desired position on the map. This will automatically populate the Latitude and Longitude fields with the appropriate coordinates.

 

        If the desired location is not visible on the map, you can navigate to it using the following techniques:

o   To move / pan the map, click and drag the mouse to a new location.

o   To zoom in/out of the map, either use the zoom button in the bottom left corner of the map or use the scroll function on the mouse or trackpad of your machine.

 

        Once all fields are populated, click the Submit Question button to upload the information to the database. If successful, the following alert will be displayed:

Displaying Existing Questions:

        Clicking the Show Questions button will display any existing questions as orange markers.

 

        It is advised to do this after every successful question submission to ensure the location is as intended.

 

        Clicking the question markers on the map will display the information inserted in the Location Name and Question fields of the form upon creation. An example can be seen below.

 

 

 

 

 

Server Related Errors:

 

        If the user experiences the following error when loading the web page, the phonegap server not running, the following will be displayed in the web browser.

 

        If the user clicks the Show Questions button and no orange markers are loaded on the map, it can be for two reasons:

o   There are not any questions in the database to add to the map.

o   The httpServer is not running.

        If any of the above occur, cancel and restart each server and the web application should work as expected.

 

Additional Notes:

 

        Currently, the web application does not support deleting or editing of questions.
